## Full description draft

### zh-TW

Lerna 是一個以專注學習為核心的 Android 讀書 App。你可以用它安排每日讀書節奏、進行專注計時、管理學習卡片與筆記，並用統計頁面回顧自己的學習進度。

重點功能：

- 專注計時與 study cycle 模式
- 任務、讀書區塊與倒數規劃
- Learn 模組：卡片、筆記、主動回想
- Stats 與個人學習追蹤
- 本機優先、離線可啟動
- 手動備份匯出 / 匯入，方便在瀏覽器、PWA 與 Android app 間轉移資料

### en

Lerna is an Android study app built for focused learning. It helps you plan daily study sessions, run focus timers, manage flashcards and notes, and review your progress in one calm interface.

Key features:

- Focus timer with study-cycle support
- Planner for tasks, study blocks, and countdowns
- Learn module with decks, notes, and active recall
- Stats and progress tracking
- Local-first behavior with offline launch support
- Manual backup export/import for moving data between browser, PWA, and Android
